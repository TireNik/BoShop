# BoShop - Микросервисный e-commerce платформа

BoShop - это полноценная электронная коммерция, построенная на микросервисной архитектуре с использованием современных технологий. Проект включает в себя все необходимые компоненты для функционирования интернет-магазина.

## Архитектура

Проект состоит из следующих микросервисов:

- **API Gateway** - единая точка входа для всех клиентов
- **Auth Service** - аутентификация и авторизация пользователей
- **User Service** - управление пользователями
- **Product Service** - управление товарами и каталогом
- **Inventory Service** - управление складскими остатками
- **Order Service** - обработка заказов
- **Notification Service** - отправка уведомлений

## Технологический стек

- **Spring Boot** - основа для всех микросервисов
- **Spring Cloud Gateway** - API шлюз
- **Spring Security + JWT** - аутентификация и авторизация
- **PostgreSQL** - реляционная база данных
- **MongoDB** - документоориентированная база данных
- **Kafka/Redpanda** - обмен сообщениями между сервисами
- **Redis** - кэширование и хранение refresh токенов
- **Docker + Docker Compose** - контейнеризация и оркестрация

## Основные возможности

### Аутентификация и авторизация
- Регистрация и вход через JWT токены
- Поддержка OAuth2 (Google)
- Ролевая модель (USER, MANAGER, ADMIN)
- Смена ролей пользователями с правами ADMIN

### Управление пользователями
- Получение информации о текущем пользователе
- CRUD операции для пользователей (только для ADMIN)
- Массовое удаление пользователей

### Управление товарами
- Создание, редактирование и удаление товаров (ADMIN)
- Публикация и сокрытие товаров из каталога
- Поиск и фильтрация товаров (по названию, категории, цене)
- Просмотр детальной информации о товаре

### Управление складом
- Просмотр остатков товара
- Пополнение и списание со склада
- Создание и обновление записей об остатках

### Обработка заказов
- Создание заказов
- Добавление товаров в заказ
- Резервирование товаров при оформлении заказа
- Оплата заказов
- Отмена заказов
- Просмотр истории заказов

## Быстрый старт

1. Склонируйте репозиторий:
```bash
git clone https://github.com/TireNik/BoShop.git
cd BoShop
```

2. Настройте переменные окружения:
```bash
cp auth-service/.env.example auth-service/.env
```

Заполните файл `.env` своими значениями для OAuth.

3. Запустите проект:
```bash
docker-compose up --build
```

## Документация API

Полная документация API доступна по адресу `http://localhost:8088/swagger-ui.html` после запуска проекта.

## CI/CD

Проект настроен с использованием GitHub Actions для:
- Проверки кода через Checkstyle
- Запуска тестов с покрытием (минимум 10%)
- Сборки Docker образов
- Тестирования деплоя
- Автоматического деплоя в dev и prod среды
